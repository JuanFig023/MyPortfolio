
# Juan Figueroa Portfolio

A personal software engineering portfolio built to showcase my background, experience, technical skills, and full-stack projects.

Live site: [juanfig023.github.io/MyPortfolio](https://juanfig023.github.io/MyPortfolio)

## Overview

This portfolio highlights:

- my background as a software engineer with experience in full-stack development
- professional experience across software engineering and data analytics
- selected solo and team projects
- a responsive UI designed to work across desktop and mobile

The site is built as a lightweight static portfolio using HTML, CSS, and JavaScript, with Tailwind CSS loaded via CDN for styling.

## Features

- responsive landing page with hero, about, experience, skills, and projects sections
- mobile navigation menu
- animated skills carousel
- dynamically rendered project cards from a central JavaScript data file
- project detail expand/collapse sections
- social and portfolio links in the navigation and footer

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Tailwind CSS CDN
- Google Fonts

## Project Structure

```text
MyPortfolio/
|-- index.html
|-- README.md
|-- css/
|   `-- styles.css
|-- js/
|   |-- nav.js
|   |-- projects.js
|   `-- skills.js
`-- images/
```

## Key Files

- `index.html` - main page structure and content sections
- `css/styles.css` - custom styles, layout rules, animations, and component styling
- `js/nav.js` - mobile navigation toggle behavior
- `js/skills.js` - skills carousel data and rendering
- `js/projects.js` - project data and project card rendering logic

## Running Locally

Because this is a static site, you can run it locally in a few simple ways:

1. Open `index.html` directly in your browser.
2. Or use a local development server such as VS Code Live Server for a better preview experience.

## Deployment

This portfolio is deployed with GitHub Pages.
